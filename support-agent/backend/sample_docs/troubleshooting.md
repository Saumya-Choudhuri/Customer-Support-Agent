# Troubleshooting Guide

## App not loading
1. Clear browser cache (Ctrl+Shift+Delete)
2. Try a different browser
3. Disable browser extensions
4. Check our status page at status.example.com

## Upload failing
- Maximum file size: 100 MB
- Supported formats: PDF, DOCX, PNG, JPG, MP4
- Check your internet connection
- If error persists, try compressing the file

## Payment declined
- Verify card details are correct
- Ensure sufficient balance
- Check if your bank is blocking international transactions
- Try UPI or Net Banking as an alternative

## API rate limits
- Free plan: 100 requests/day
- Pro plan: 10,000 requests/day
- Enterprise: Unlimited
- If exceeded, you'll receive a 429 error. Wait until midnight IST for reset.
